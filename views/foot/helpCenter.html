个人会员常见问题

一、注册及会员账号常见问题
  1. 如何成为猎评网会员？
    2. 注册成为猎评网会员很简单，只需一步：
    填写登记作为登录账号的个人手机号或则E-mail，快速注册完成，接下来就可以直接去填写简历找工作，也可以进入会员中心搜索及订阅心仪的职位。立即注册
  2. 忘记密码或用户名怎么办？
    3. 猎评网提供了快捷找回密码功能，只需提供您在注册时登记的手机号码或E-mail地址，我们会自动将重置过的密码发送至您的手机或邮箱。此功能同样适用于忘记用户名的情况。忘记密码 如果收到重置密码还是不能登录，或多次登录错误帐户被限制，请联系猎评网客服：
  3. 如何修改密码？
    4. 登录个人会员中心，找到用户管理页，即可重新设置新密码。修改密码
  4. 注册时发现输入的邮件地址已经被使用怎么办？
    5. 这说明你已经注册过本网站了。请试着输入用户名和密码登录你的会员中心。如果忘记了用户名和密码，可以点击忘记密码，在注册邮箱中查询新密码，登录后修改完善简历重设密码，当然也可以用其他邮箱重新注册。
二、简历常见问题
  1. 注册后如何完成自己的简历信息？
    2. 当您完成注册后，可以按照页面指示，输入相关信息，就可以完成一份电子简历。请按照页面提示完成简历，这样才能保证简历内容的完整性，提升简历的吸引力。 另外，我们的简历是即填即存式的，所以每当您填写完成一个部分，务必记得要随时进行保存操作，您的信息才能存入我们的数据库中。 如果您希望简历更加个性化，可以将制作好简历直接上传到附件简历中。
  2. 如何上传个人照片？为什么上传的照片没有显示？
    3. 在编辑个人标准简历时，专门有一项上传照片的功能供用户选择使用。照片大小控制在30k以内，图片宽度*高度最好为100*120像素，Chr支持jpg、gif、png格式。个人照片需审核修改后才能显示，一般为隔天进行显示。
  3. 如何上传我的学历证书和资格证书？
    4. 在填写完整简历时，可以在教育背景和证书中列出所获学历和证书，但Chr目前暂不支持直接上传证书。请不要将证书上传到照片栏。
  4. 可以创建多少份简历？
    5. 每位注册用户可以创建1份标准简历，和上传一份附件简历。但记住，只有一份标准简历可以设置为完全公开，供招聘单位进行搜索。请一定要保证这份公开简历的完整性和质量，以便获得更多的工作机会。
三、个人信息安全与费用问题
  1. 关于个人信息的保密？
    2. 当您选择简历开放功能时，您已经同意猎评网企业会员搜索到并查看您的简历。 如果您不愿意简历被搜索查看，您可以选择简历隐藏功能。但仍可以通过猎评网网个人会员管理系统注册完善的电子简历，并自行查找工作职位、订阅工作职位、在线应聘。
  2. 个人用户应聘时否收费？
    3. 猎评网针对个人用户的应聘、简历管理、职位订阅等，均为免费服务。我们会代理企业进行某些职位的招聘，这类招聘针对个人用户依然是免费服务。如果您在猎评网遇到需付费的应聘情况，请投诉至：400-710-4000

  1. 忘记密码怎么办？
    2. 如果忘记了密码，请通过注册的电子邮箱找回。
  2. 为什么一定要填写公司信息？
    3. 清晰明了的公司介绍有利于求职者了解企业情况，为招聘企业吸引到适合人才，减少冗余简历的处理。在Chr系统中，公司介绍为必填项。
  3. 怎样上传企业Ｌｏｇｏ、照片？
    4. 进入企业管理系统后，在帐户管理中，可以发布、修改企业介绍，也可以直接上传企业Logo、照片。Logo最佳大小为100*50，大小不得大于50K；图片最佳大小为187*145，企业可以上传两张形象照；
